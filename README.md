# 🎂 Mushtaribonu — 9 Yosh Tug'ilgan Kun Tabrik Sayti

Mushtaribonu uchun maxsus tayyorlangan interaktiv tug'ilgan kun tabrik veb-sayti.

## ✨ Imkoniyatlari

- 🎁 Sovg'a ochish animatsiyasi
- 🎆 6 xil vizual effektlar (yuraklar, salut, oltin, sharlar, gullar, konfetti)
- 📊 9 yillik hayot statistikasi
- ⏰ Keyingi tug'ilgan kungacha countdown
- 🔮 Burj bashorati
- 🎰 Omad g'ildiragi
- 🃏 Xotira o'yini
- 💝 Yuraklar tutish o'yini
- 📸 3D galereya
- 🎵 Musiqa pleyer
- 🎁 9 ta virtual sovg'a
- 💌 Maxfiy xat
- 📝 Mehmonlar kitobi
- 🏆 Yutuqlar tizimi
- 🌟 Tilaklar generatori

## 🚀 Ishga tushirish

Loyiha bitta `index.html` fayldan iborat. GitHub Pages orqali joylashtirish uchun:

1. GitHub'da yangi repository yarating
2. `index.html` faylni repoga yuklang
3. **Settings → Pages → Source** bo'limida `main` branch tanlang
4. Sayt tayyor!
